# Technical Content Evaluation Framework

## Documentation Wrapper Score (0-100)
- **External links only**: -40 points
- **No starter code/solutions**: -30 points
- **Missing local files**: -20 points
- **"Under construction" placeholders**: -10 points

## Grading Scale
- **90-100 (A)**: Self-contained, real exercises, pedagogical flow.
- **70-89 (B)**: Hybrid content, mostly teaching, some external dependencies.
- **50-69 (C)**: Documentation wrapper with minor teaching elements.
- **<50 (F)**: Resource index/Link collection only.

## Content Audit Methodology
1. **Real**: Commands to run, code to write, success criteria.
2. **Partial**: Steps provided but missing validation.
3. **Aspirational**: Vague bullet points with no guidance.

## Visual Aid Strategy
- Suggest **Flowcharts** for logic paths.
- Suggest **Sequence Diagrams** for API timing.
- Suggest **Architecture Diagrams** for resource mapping.
